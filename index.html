    03 — Deep

    Hold for Strength

    Hold for Strength is a paper-trading system I built from scratch to test one idea: buy momentum names that also hold up fundamentally, then let a fixed set of exit rules make the selling decision instead of me. I run it in an Excel workbook alongside a trade blotter I wrote, and I log every entry and exit against live prices. The point is not the return. The point is that the rules are written down before the trade, so I can tell afterwards whether a loss came from the idea or from breaking my own process.
